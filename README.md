# OHHM App 2

App de análisis de técnica de atletismo desarrollada por Oscar Herrera Muro.

## Descripción

Esta aplicación permite analizar la técnica de atletas, brindando métricas y observaciones para mejorar el rendimiento. Es ideal para entrenadores y atletas que desean llevar un control detallado de sus entrenamientos.

## Tecnologías

- JavaScript
- HTML
- CSS
- Despliegue: [Vercel](https://ohhm-app-2.vercel.app)
